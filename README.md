# 小雨办公工具

一款基于 PySide6 开发的办公自动化工具，专注于符号处理、表格编辑和实用工具集成。

## 功能特性

### 🎨 符号处理
- 多种常用符号（引号、括号、书名号等）
- 4种插入位置（头部、尾部、两端、分隔）
- 符号叠加功能
- 中文/英文符号切换
- 自定义符号
- 公式助手
- 实时预览

### 📊 表格编辑
- 打开、编辑、保存 Excel 文件
- 数据查找、替换
- 行/列管理（插入、删除、冻结）
- CSV导出
- 撤销/重做支持
- 大文件优化加载
- 公式编辑栏
- 单元格样式保持

### 📝 文本处理
- 文本格式化
- 文本转换
- 文本编辑
- 符号处理集成

### 🔧 工具箱
- **剪贴板历史**：记录剪贴板内容，快速调用
- **批量文件处理**：重命名、分类、改后缀
- **图片工具**：裁切、压缩、格式转换
- **压缩解压**：支持文件夹递归
- **屏幕截图**：全屏/区域/窗口截图
- **图片OCR**：内置Tesseract引擎，支持中英文识别
- **文本格式化**：文本处理和格式转换
- **格式互转**：Excel/Word格式互转
- **数据备份**：配置文件备份与恢复

### 📚 函数教程
- 常用函数说明（求和、平均值、查找等）
- 卡片式布局，清晰直观
- 搜索功能

### ❓ 问题解决
- 各模块使用帮助
- 常见问题解答
- 搜索功能

### 🎭 主题系统
- 6种内置主题
  - 现代蓝色（免费版）
  - 活力橙色（Pro版）
  - 商务深蓝（Pro版）
  - 护眼豆沙绿（Pro版）
  - 樱花粉（Pro版）
  - 暗黑模式（Pro版）
- 实时主题切换
- 全局样式覆盖

### 💎 免费版/Pro版
- **免费版**：符号处理、函数教程、问题解决
- **Pro版**：全部功能解锁，包括表格编辑、文本处理、工具箱、主题系统

## 快速开始

### 安装依赖
```bash
pip install -r requirements.txt
```

### 启动程序
```bash
python main.py
```

### 启动Pro版（临时）
```bash
python main.py pro
```

## 技术栈
- **UI框架**：PySide6 6.11.1
- **表格处理**：openpyxl 3.1.5
- **文档处理**：python-docx 1.2.0
- **图像处理**：Pillow 12.2.0, opencv-python 4.13.0.92
- **OCR识别**：pytesseract 0.3.13
- **加密**：cryptography 48.0.0
- **图标**：QtAwesome 1.4.2
- **中文分词**：jieba 0.42.1

## 系统要求
- **操作系统**：Windows 10/11
- **Python**：3.10+（开发环境）
- **内存**：建议 4GB 以上
- **磁盘空间**：至少 500MB
